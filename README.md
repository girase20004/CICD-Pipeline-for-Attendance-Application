# CICD-Pipeline-for-Attendance-Application
Built a CI/CD pipeline for an Attendance Application using AWS CodePipeline and EC2/S3 to automate deployment and updates.


## 🎯 Purpose
Automatically deploy the attendance application on every code update.

## 🧰 AWS Services Used
- AWS CodePipeline
- Amazon EC2 / Amazon S3

## 📌 Project Overview
This project demonstrates a CI/CD pipeline for an attendance application using AWS services. 
The application is automatically deployed whenever code changes are pushed.

## 🚀 Features
- Automated deployment
- Continuous integration
- Continuous delivery
- Faster updates
